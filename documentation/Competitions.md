# Competiciones

Esta es la fase de programacion de las Competiciones.
Los programas de emparejamientos (Vega, Swiss-Perfect, etc) seran los que surtan los datos para el historial de este programa.  Este proyecto no es su finalidad.

La finalidad es la de generacion de competiciones y cuadres de competiciones.

Una de las ideas para el furturo es la de incorporar los ficheros de los programas de emparejamiento que tengan una codificacion abierta.
