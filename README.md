# FedeChess
Aplicacion para administrar federaciones de ajedrez y sus delegaciones. Asi como todas las dependencias que estan subordinadas a la Federacion.  

### Administracion y Gestion

- Junta Directiva 
- Delegaciones
- Tesoreria y Administracion

### Area deportiva 

- Federados  
- Arbitros
- Monitores
- Entrenadores

### Competiciones  

- Torneos Provinciales Individuales  
- Torneos por Equipos  
- Torneos Andaluces Individuales  
- Torneos Andaluces por equipos
