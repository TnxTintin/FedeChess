# FedeChess
Aplicacion para administrar federaciones de ajedrez y sus delegaciones. Asi como todas las dependencias que estan subordinadas a la Federacion.  

### Administracion y Gestion

- Junta Directiva 
- Delegaciones
- Tesoreria y Administracion
- Clubes

### Area deportiva 

- Federados  
- Arbitros
- Monitores
- Entrenadores

### Competiciones  

- Torneos Provinciales Individuales y por Equipos
- Torneos Andaluces Individuales  y por Equipos

- Copas Provinciales Individuales y por Equipos
- Copas Andaluzas Individuales y por Equipos
